My-Chat Widget Test Page

This page simulates a client's website with the embedded chat widget.

The widget is loaded with data-widget-key="demo-widget-key" pointing to http://localhost:3300

Click the blue chat bubble in the bottom-right corner to start a conversation.

Quick start

  1. docker-compose up -d (from project root)
  2. cd backend && npx prisma migrate dev --name init && npm run prisma:seed
  3. npm run start:dev
  4. Open this file in a browser
