Gotowy skrypt do wklejenia na stronę klienta (localhost)

Uruchom najpierw frontend (cd frontend && npm start) oraz backend (cd backend && npm run start:dev). Następnie wklej poniższy blok przed zamknięciem </body> na dowolnej stronie (np. inny projekt na localhost).

URL skryptów: http://localhost:5200/widget/loader.js (frontend musi działać na porcie 5200).
Backend czatu: http://localhost:3300 (serverUrl w init). 
<!-- My-Chat Widget -->
<script>
(function(w,d){
  if(w.MyChat)return;
  var q=[];
  w.MyChat=function(){q.push(arguments)};
  w.MyChat._q=q;
  var s=d.createElement("script");
  s.async=true;
  s.src="http://localhost:5200/widget/loader.js";
  d.head.appendChild(s);
})(window,document);

MyChat("init",{widgetKey:"demo-widget-key",serverUrl:"http://localhost:3300"});
</script>

Po załadowaniu strony możesz wywołać w konsoli: MyChat.open(), MyChat.close(), MyChat.destroy().
